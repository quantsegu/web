How ratings are calculated

Every category uses a 0–100 score where higher is better. Scores are fixed reference values for 2024/2025 conditions (not live data). They are meant for side-by-side shortlisting, not as precise forecasts.

Weighted country score = (wtax×Tax + wliving×Living + … + wquality×Quality) ÷ (wtax + wliving + … + wquality)

Slider weights (0–100) are relative importance only. A weight of 80 vs 40 means that category counts twice as much in the average, not that it is measured on a different scale.
