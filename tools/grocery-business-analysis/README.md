# Grocery Business Analysis

Source for the **Grocery Business Analysis — CEE Fulfillment** wealth tool. Based on [quantsegu/Grocerybusinessanalysis](https://github.com/quantsegu/Grocerybusinessanalysis), extended for Central & Eastern European fulfillment hubs (Poland, Romania, Czech Republic, Hungary) instead of Western European shipping.

## Develop

```bash
cd tools/grocery-business-analysis
npm install
npm run dev
```

## Build (outputs to `public/wealth-tools/grocery-business-analysis/`)

```bash
npm run build:grocery-business-analysis
```

From the repository root, or `npm run build` which runs this before the main site build.
